## Description

This application is the frontend for the productfinder application

## 1.Install

### npm

```
npm i
or
npm i --legacy-peer-deps
```

### yarn

```
yarn install
```

## 2.Start

```sh
npm start
or
yarn start
```

## 3.Build

```sh
npm run build or yarn build
```
